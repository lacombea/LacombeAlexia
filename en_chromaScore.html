About

Play

Play with the ←↓→ arrows, or the ASD (qwerty) or QSD (azerty) keys.
Left is cyan, Down is yellow, Right is magenta.

My role

I participated in the development and programming of this game. Here are the main tasks I took care of during the project:

Implementation of menus and parameters

I implemented the navigation between the different menus and scenes of our game and also implemented the options.
I discovered FMOD and the way to manage the sound of our game via its API.

  • Main menu

  • Options menu

  • Pause menu

Implementation of backups

To anchor our game in the Arcade style, we wanted to have the possibility to display a scoreboard. The data we will save will be for each music a table containing the 10 best scores. The score will contain a name, a number of points and a rank.
In addition, we will also save the sound settings so that the user can keep them between several games.
To do this, I set up a backup system using a json file.

  • Saved data

  • Class Score

  • Display of scores

Build WebGL

I also took care of creating the WebGL build of the game. It is at this stage that we decided to add a loading screen allowing FMOD to initialize itself before launching the game. Without this screen, the sound of the game was only launched by changing the menu.
